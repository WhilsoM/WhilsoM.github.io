1. на странице достопримечательности должен быть реализованы: 
  ✅ - пагинация с сервера по 10 шт, при нажатии на кнопку следующей страницы отправляется запрос на сервер и подгружаются следующие 10 страниц, во время загрузки отображать loader. done
  
  ✅ - сортировка - сделать сортировку по популярности, сначала популярные или сначала непопулярные, по цене, по алфавиту. При переключении сортировки должен отправляться запрос на сервер и данные должны сортироваться именно там, а не в js коде. 
  При сортировке кол-во достопримечательностей не уменьшается, меняется только их порядок отображения.
  ✅ - фильтрация - фильтровать по району, типу. При переключении фильтров должен отправляться запрос на сервак и данные должны фильтроваться именно там, а не в js. Фильтры могут юзаться совместно с сортировкой и поиском. 
  ✅ - поиск должен искать достопримечательность по названию, юзая сервак, то есть при введении что-либо в поисковую строку отправляется запрос на сервер с параметрами.

2. открытие страницы одной достопримечательности.
  - должно быть реализовано через Url, обычно для открытия конкретной записи используют подобный вид https://domain.com/users/1 - где 1 это ID записи. На страницу с достопримечательностью также необходимо добавить несколько фотографий,
  
  при нажатии на которые – фото открывается на весь экран и их можно переключать, как галерея.

  -На странице достопримечательности должны появится отзывы, для того чтобы оставить отзыв
необходимо заполнить минимум два поля: имя и текст отзыва. Вы можете добавить дополнительные
поля на свое усмотрение, при отправке формы поля должны валидироваться. При отправке формы
должен быть отправлен запрос на сервер для добавления отзыва. Отзывы должны подгружаться с
сервера, должна быть возможность удалить любой из отзывов.
